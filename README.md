# Proje Başlığı
Credit card fraud

## Açıklama
Bu proje, kredi kartı bilgilerini yönetmek için bir uygulama geliştirmeyi hedeflemektedir. Uygulama, kullanıcıların kredi kartı bilgilerini eklemelerine, düzenlemelerine ve takip etmelerine olanak tanır.

## Özellikler
- Kredi kartı ekleme ve düzenleme
- Harcama takibi
- Güvenli veri saklama

## Teknolojiler
- Dart
- Flutter
- SQLite
- Provider
- Encrypt
